• ipDISK 주소 관리
    • 사용하고자 하는 ipDISK 주소 및 이메일 주소를 입력 후 적용 버튼을 클릭합니다.
    • ipDISK 상태
      • [정상 등록] - 설정한 ipDISK 주소를 사용하여 각각의 서비스를 접속할 수 있습니다.
      • [다른 사용자의 호스트이름] - ipDISK 주소를 다른 이름으로 설정해야 합니다.
      • [연결 실패] - 인터넷 연결을 확인 후, 재시도 해야합니다.

  • ipDISK(FTP) 서비스 설정
    • ipDISK 주소를 사용하여 일반 FTP클라이언트 접속 및 ipDISK 모바일APP 접속이 가능합니다.
    • 포트 : 접속 시 사용될 TCP포트 번호입니다.
    • 문자셋 : 접속 시 사용될 클라이언트의 문자셋을 정의합니다.
    • 사용자 ID/암호 : 접속 시 사용될 클라이언크의 ID/암호 입니다.
    • HTTP 포트: ipDISK Mobile앱에서 사용되는 미디어 서비스 포트입니다.
    • 해당 서비스는 USB저장장치가 연결된 이후에 동작합니다.
    • 접속 프로그램 - [ PC용 ipDisk Drive 프로그램 다운로드 하기 ]

  • Cloud백업 서비스 설정
    • ipTIME에서 제공하는 Cloud백업 프로그램을 이용하여, "PC의 파일을 USB하드디스크로" 또는 "USB하드디스크의 파일을 PC로" 백업할 수 있습니다.
    • 파일 백업 시 변경된 파일만 복사합니다.
    • Cloud백업 서비스는 USB저장장치가 연결된 이후에 동작합니다.
    • Cloud백업 서비스는 rsync를 기반으로 동작합니다.
    • 접속 프로그램 - [ PC용 ipTIME Cloud백업 프로그램 다운로드 하기 ]

  • 윈도우 파일 공유 서비스 설정
    • USB저장장치의 파일을 윈도우 파일 공유 서비스를 통해 공유합니다.
    • \\[IP주소] 또는 \\[서버이름] 으로 윈도우 파일공유를 사용할 수 있습니다.
    • 예> \\192.168.0.1 또는 \\myserver
    • 해당 서비스는 USB저장장치가 연결된 이후에 동작합니다.

  • 토렌트 설정
    • 토렌트파일을 이용하여 파일을 다운로드 할 수 있습니다.
    • [토렌트 웹UI]에 접속하여 사용하거나 [토렌트 접속기]를 다운로드하여 PC에 설치 후 사용할 수 있습니다.
    • 해당 서비스는 USB저장장치가 연결된 이후에 동작합니다.
    • 해당 기능은 일부 제품에 포함되어 있지 않을 수 있습니다.

  • 미디어 서버
    • 미디어 폴더내에 저장된 미디어 파일들을 DLNA표준 프로토콜을 통해 서비스합니다.
    • 스마트TV 또는 스마트폰등의 플레이어 장치와 연동하여 사용할 수 있습니다.
    • 해당 서비스는 USB저장장치가 연결된 이후에 동작합니다.
    • 해당 기능은 일부 제품에 포함되어 있지 않을 수 있습니다.

  • Apache서버
    • Apache 웹서버를 구동할 수 있습니다.
    • PHP를 지원합니다.
    • Apache서버에 대한 자세한 정보는 http://www.apache.org를 참고합니다.
    • ServerRoot 는 apache자체 config파일 및 로그/임시파일/Session파일 등이 저장되는 공간입니다.
    • ServerRoot폴더내의 httpd.conf.templ 를 변경하면 사용자가 원하는 httpd.conf를 사용할 수 있습니다.
    • DocumentRoot 는 웹서버의 Root폴더 입니다.
    • [Plug-in APP관리] - [Plug-in App검색/설치]에서 다양한 App을 다운로드하여 실행할 수 있습니다.
    • 해당 기능은 일부 제품에 포함되어 있지 않을 수 있습니다.

  • MySQL서버
    • MySQL서버를 구동합니다.
    • DB폴더는 MYSQL 서버에서 사용하는 DB 데이터 저장 위치 입니다.
    • 서버문자셋을 통해 MYSQL DB 저장 시 사용할 문자셋을 지정할 수 있습니다.
    • 클라이언트에서 서버문자셋 사용 옵션을 통해 클라이언트에서 서버 접속 시, 클라이언트가 서버의 문자셋을 따를 것인지 여부를 설정할 수 있습니다.
    • Max Allowed Packet은 MYSQL 서버에서 허용하는 최대 Packet 사이즈며, 기본값은 1MB입니다.
    • 해당 기능은 일부 제품에 포함되어 있지 않을 수 있습니다.

  • USB테더링
    • USB테더링 기능을 사용하면, 스마트폰을 통한 인터넷을 사용할 수 있습니다.
    • 공유기와 스마트폰을 USB케이블로 연결 하고, 스마트폰의 [USB테더링기능]을 활성화한 후 사용 가능합니다.
    • 안드로이드 계열 스마트폰을 지원합니다.(일부 호환되지 않는 기종이 있을 수 있습니다.)
    • 스마트폰이 3G/LTE등으로 연결된 경우, 요금제에 따라 데이터 요금이 발생할 수 있습니다.
    • 공유기의 WAN포트가 연결되어 있는 상태에서는 동작하지 않습니다.
    • 해당 기능은 일부 제품에 포함되어 있지 않을 수 있습니다.

  • URL 서비스 설정
    • 일반 웹브라우저를 이용하여 USB저장장치의 파일을 공유할 수 있습니다.
    • [접속URL]을 이용하여 파일을 검색하고 다운로드 할 수 있습니다.
    • 해당 서비스는 USB저장장치가 연결된 이후에 동작합니다.

  • 네트워크 프린터 서버 설정
    • 네트워크 프린터 서버 기능을 통해 공유기에 연결된 USB 프린터를 네트워크 프린터 방식으로 이용할 수 있습니다.
    • 해당 기능은 일부 제품에 포함되어 있지 않을 수 있습니다.
  • 접속자 상태
    • 현재 연결되어 있는 클라이언트의 연결정보를 나타냅니다..
