• 개 요
    • 이 기능을 이용하면 한개의 AP로 여러개의 무선 네트워크를 구성할 수 있습니다.
    • 기본 무선 네트워크는 [ 무선 설정/보안 ] 에서 설정된 무선 네트워크를 의미합니다.
    • 각각의 무선 네트워크에 독립적인 인증/암호화 설정이 가능합니다.
    • 모드,채널,지역 설정은 기본 무선 네트워크와 동일합니다.
  • 사용 방법
    • [새 무선 네트워크] 를 클릭합니다.
    • 인증 및 암호화를 선택한 후, 네트워크 암호를 입력합니다.
    • [추가] 버튼을 클릭합니다.
  • 설정
    • 무선 네트워크 이름
      • 추가될 무선 네트워크 이름을 입력합니다.
    • 네트워크이름 알림
      • 추가될 무선 네트워크를 숨길 수 잇습니다.
      • 이 기능을 사용하지 않을 경우 네트워크 이름을 알아야만 접속할 수 있습니다.
    • 통신 정책
      • 모든 통신 허용 - 본 공유기를 통한 인터넷 및 내부 네트워크로의 통신을 허용합니다.
      • WAN포트를 통한 인터넷 통신만 허용 - 인터넷 통신만 허용합니다. 내부 네트워크로의 통신은 허용하지 않습니다.

    • 암호화 방법
      • 보안 암호화 설정을 통하여 허가된 사용자만이 무선랜을 통하여 접속할 수 있습니다.
      • 인증 및 암호화 방법
        • WPA2PSK + AES : 가장 강력한 암호화 방법으로 사용하기를 권장합니다.
        • TKIP 또는 WEP을 통한 암호화를 사용할 경우 802.11 무선 표준에 의거하여 11g모드로 동작합니다.
        • TKIP/AES 암호화를 선택하였을 경우 8글자 이상의 암호를 입력해야합니다.
        • WEP 암호화는 매우 취약한 암호화 방법이므로 사용하지 않을것을 권장합니다.
