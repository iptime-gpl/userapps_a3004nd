• 동적 IP 방식
    • FTTH, 광랜, 케이블 모뎀, VDSL, IP ADSL(자동접속기능 내장)
    • MAC 주소 변경 : 인터넷 서비스 제공자가 MAC주소를 제한 할 경우에 사용합니다.
    • 사설 IP 할당 허용하기 : 사설 IP를 할당 받을 수 있게합니다.( 172.16.x.x, 192.168.x.x, 10.x.x.x )


  • PPPoE 방식
    • 일반 ADSL 사용자
    • 하나로 통신의 경우 사용자계정(ID) 입력시 'ID@hanaro' 형식으로 입력하셔야 합니다.


  • 고정 IP 방식
    • 전용선 및 MyIP, Multi IP 사용자


  • MTU
    • 네트워크에서 전송될 수 있는 패킷의 최대 크기를 입력 합니다.
  • 수동으로 공유기의 DNS 서버 설정
    • DNS주소를 수동으로 지정할 수 있습니다.
    • 수동으로 지정을 하지 않으면 인터넷에서 받아온 값을 사용합니다.
