• 로그인 암호 설정
    • 설정 웹 서버 접속 시 사용되는 관리자의 계정 및 암호를 설정할 수 있습니다.
    • 시스템 관리 -> 설정백업/복구 -> 초기 설정 복원을 실행하는 경우에는 설정된 암호는 없어집니다.
    • 세션 방식을 사용할 경우에는 계정 및 암호를 삭제할 수 없습니다.

  • 인증방법
    • 기본방식: HTTP Basic인증을 사용한 방식
    • 세션방식: 보안이 강화된 인증방식
    • 세션방식-로그인시 문자열 입력: 로그인시 무작위로 생성된 코드(영문소문자 5글자)를 입력하게하여 각종 인증관련 공격을 방지합니다.

  • 관리자 E-Mail 설정
    • 시스템에 관련된 정보를 받을 E-mail 주소를 입력합니다.
    • 사용하고 있는 SMTP 메일 서버를 입력합니다.
