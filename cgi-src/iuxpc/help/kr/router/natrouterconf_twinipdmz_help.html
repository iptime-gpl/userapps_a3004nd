• DMZ
    • 지정된 내부 IP주소의 PC로 모든 포트를 개방합니다.
    • 단, 포트포워드에 설정된 포트는 제외됩니다.

  • TWIN IP
    • 지정된 MAC주소의 PC가 공인 IP주소를 사용하도록 합니다
    • 즉 Twin IP 기능이 설정된 PC의 경우에는 내부 사설 IP주소가 아닌 공인 IP주소를 가지고 공유기를 사용하지 않는 환경과 동일하게 인터넷을 사용하실 수 있는 기능입니다.
    • 체크박스를 체크하시면 현재 접속한 PC의 하드웨어 주소가 설정되고, '적용'버튼을 클릭하면 설정이 완료됩니다.

  • 중단
    • DMZ과 TWIN IP기능을 모두 사용 하지 않습니다.
