• VPN(PPTP) 서버 설정
    • 동작 설정 : VPN(PPTP) 서버에 대한 동작 상태를 설정할 수 있습니다.
    • 암호화(MPPE) : VPN(PPTP) 서버 대한 동작의 암호화를 설정할 수 있습니다.
  • VPN(PPTP)계정 설정
    • VPN 접속시 요구되는 계정,암호,할당될 IP 주소를 설정 합니다.
    • 할당될 IP주소는 내부 IP와 동일한 서브넷의 IP로 설정 합니다.
    • 할당될 IP주소는 동적IP 주소범위와 겹치지 않게 합니다.
    • 연결 상태 : 해당 계정의 VPN연결 접속 상태
    • 연결 끊기 버튼 : VPN연결을 강제 종료
    • 삭제 버튼 : VPN연결 구성을 삭제
    • VPN 접속 계정은 최대 5개 까지 설정 가능합니다.
    • VPN 연결은 동시에 최대 5개 까지 가능합니다.
    • 같은 계정으로 다수의 연결을 할 수 없습니다.
    • *관리자 계정 및 암호를 설정하지 않고 VPN계정 추가 시, 보안코드를 입력해야 합니다.
  • 설정 예
      • 1> 네트워크 관리 -> 내부 네트워크 설정 -> 동적 IP주소범위 변경 (192.168.0.2 ~ 240)
      2> 특수기능 -> VPN 서버설정
        VPN접속 계정 : iptime
        VPN접속 암호 : iptimeisbest
        IP 주소 : 192.168.0.241
