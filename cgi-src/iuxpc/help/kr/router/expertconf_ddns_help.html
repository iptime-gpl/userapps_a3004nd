• DDNS의 기능
    • DDNS는 일반 인터넷회선(유동 IP)을 이용하여 공유기 하단에서 서버를 운영 할 수 있는 기능입니다.
    • 즉, DDNS를 사용하시면 유동 IP 주소를 도메인 이름으로 자동 변경하여 매번 공유기의 외부 IP 주소를 확인하지 않아도 편리하게 서버운용을 할 수 있습니다.
  • DDNS 동작/계정 설정
    • ipTIME DDNS 를 사용하시게 되면 별도의 등록절차가 필요하지 않습니다.
    • ipTIME DDNS 사용시에는 호스트이름이 추가되면 자동으로 계정이 추가됩니다.
    • *관리자 계정 및 암호를 설정하지 않고 DDNS를 설정 시, 보안코드를 입력해야 합니다.
