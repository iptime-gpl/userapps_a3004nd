• 인터넷 사용제한
    • 특정 IP나 PC의 인터넷 사용을 제한하는 기능입니다.
    • 최대 200개의 규칙을 입력할 수 있습니다.

    • 규칙이름
      • 임의로 입력가능하며 구분이 쉽도록 입력합니다.(최대 127자)
    • 규칙 비활성화
      • 사용자 스케줄에 관계없이 규칙을 정지시킵니다.
    • 방향
      • 내부->외부 : 공유기에 연결된 내부의 PC에서 외부로 통신
      • 내부<-외부 : 외부에서 공유기에 연결된 내부의 PC로 통신
      • 내부<->외부 : 모든 방향 통신
    • 프로토콜
      • Protocol Type (ALL/TCP/UDP/ICMP/GRE/TCPUDP)을 선택합니다.
      • 'ALL'은 모든 프로토콜을 의미합니다. 'ALL', 'ICMP', 'GRE' 선택시에는 포트번호를 입력할 수 없습니다.
    • 포트번호
      • 'TCP', 'UDP', 'TCPUDP' 프로토콜을 선택한 경우, 포트 번호를 단일 혹은 범위로 입력합니다.
    • 내부 주소
      • 내부IP 주소 : 내부PC의 IP 주소
      • 모든 내부IP : 내부PC의 모든 IP 주소
      • MAC 주소 : 내부PC의 MAC 주소
    • MAC주소 검색
      • 내부 주소 선택에서 MAC 주소를 선택한 경우, 입력칸 우측에 검색 버튼이 생성됩니다.
    • 외부 주소
      • 외부IP 주소 : 외부PC의 IP 주소
      • 모든 외부IP : 외부PC의 모든 IP 주소
    • 허용/차단
      • 정의한 통신 접속 형태의 허용/차단을 선택합니다.
    • 사용자 스케줄 설정하기
      • 스케줄-요일 : 설정을 적용할 요일을 선택합니다. (선택하지 않을 경우, 자동으로 '매일'로 설정됩니다.)
      • 스케줄-시간 : 설정을 적용할 시간대를 선택합니다.(선택하지 않을 경우, 자동으로 '24시간'으로 설정됩니다.)


  • 사이트 접속 차단
    • 특정 IP나 PC의 특정 문자열을 포함하는 사이트 주소에 대한 접속을 제한하는 기능입니다.

      • MAC 주소 : 내부PC의 MAC 주소
    • 차단할 사이트 입력
      • 접속을 차단하고자 하는 사이트 주소의 문자열을 입력합니다.
